## vscode 常用插件推荐

### Beautify
快捷键 `ctrl + l`

安装这一个插件可以分别格式化 `js、css、html` 等文件

### Debugger for Chrome
在 VsCode 上调试 js 代码

### JavaScript File Header
快捷键 `window + Alt + H`

快速创建文件头信息，头信息模板可自定义

### Markdown Preview Enhanced
Markdown 文件优化显示

### open in browser
在指定浏览器中打开 html 文件

### TFS
连接 TFS 管理器，只能迁出，无法迁入

### vscode-icons
项目文件夹小图标显示

### Markdown TOC
快速为 MD 文件新建目录

### Bracket Pair Colorizer
括号 `()、[]、{}` 样式优化显示
